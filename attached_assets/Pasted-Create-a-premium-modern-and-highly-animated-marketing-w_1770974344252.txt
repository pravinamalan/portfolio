### Animations with Framer Motion
Implement these patterns:

1. Fade In on Scroll:
```jsx
<motion.div
  initial={{ opacity: 0, y: 50 }}
  whileInView={{ opacity: 1, y: 0 }}
  viewport={{ once: true }}
  transition={{ duration: 0.6 }}
>
```

2. Stagger Children:
```jsx
<motion.div
  variants={containerVariants}
  initial="hidden"
  whileInView="visible"
  viewport={{ once: true }}
>
  {items.map((item) => (
    <motion.div variants={itemVariants}>
      {item}
    </motion.div>
  ))}
</motion.div>
```

3. Hover Effects:
```jsx
<motion.button
  whileHover={{ scale: 1.05, boxShadow: "0 10px 40px rgba(0,0,0,0.2)" }}
  whileTap={{ scale: 0.95 }}
>
```

4. Counter Animation:
```jsx
<motion.span
  initial={{ opacity: 0 }}
  whileInView={{ opacity: 1 }}
  viewport={{ once: true }}
>
  {/* Use a counter component that animates from 0 to target number */}
</motion.span>
```

5. Parallax on Hero:
```jsx
const { scrollY } = useScroll();
const y = useTransform(scrollY, [0, 500], [0, 200]);

<motion.div style={{ y }}>
  {/* Content that moves slower than scroll */}
</motion.div>
```

## EXAMPLE BUTTON COMPONENT WITH ANIMATIONS

```tsx
import { motion } from 'framer-motion';

export function Button({ children, variant = 'primary', ...props }) {
  const baseStyles = "px-8 py-4 rounded-full font-semibold text-lg transition-all";
  
  const variants = {
    primary: "bg-gradient-to-r from-teal-500 to-purple-600 text-white",
    secondary: "border-2 border-navy bg-transparent text-navy hover:bg-navy hover:text-white",
  };

  return (
    <motion.button
      className={`${baseStyles} ${variants[variant]}`}
      whileHover={{ 
        scale: 1.05, 
        boxShadow: "0 20px 40px rgba(0,0,0,0.15)" 
      }}
      whileTap={{ scale: 0.95 }}
      {...props}
    >
      {children}
    </motion.button>
  );
}
```

```tsx
      >
        <img 
          src={image} 
          alt={title}
          className="rounded-2xl shadow-2xl"
        />
      </motion.div>
    </motion.div>
  );
}
```
